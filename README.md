# 🚢 Titanic Survival Analysis (타이타닉 생존자 분석)

## 1. Project Overview (프로젝트 개요)
- **Goal:** 타이타닉 탑승객 데이터를 분석하여 생존에 영향을 미치는 핵심 요인을 파악합니다.
- **Dataset:** Kaggle Titanic Dataset
- **Tools:** Python, Pandas, Matplotlib, Seaborn

## 2. Key Findings (핵심 분석 결과)
데이터 분석을 통해 얻은 주요 인사이트는 다음과 같습니다:

1.  **여성과 아이 우선 (Women & Children First):**
    - 여성의 생존율은 74%인 반면, 남성은 18%에 불과했습니다.
2.  **부의 생존 법칙 (Social Class):**
    - 1등석(Pclass 1) 승객의 생존율이 3등석보다 압도적으로 높았습니다.
3.  **가족의 규모 (Family Size):**
    - 혼자 온 승객보다 2~4명의 가족과 함께한 승객의 생존율이 가장 높았습니다.

## 3. Conclusion (결론)
타이타닉호의 생존은 단순한 운이 아니라, **성별, 사회적 지위(등급), 가족 동반 여부**에 의해 구조적으로 결정되었음을 데이터로 확인했습니다.
